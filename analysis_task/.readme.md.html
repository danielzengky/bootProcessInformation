1 所有计算任务的基类

   comm
      |--task_online

1.1 comm_analysis

  • 所有计算的基本模式一样，不同点：

    • 1 测点和结果点

    • 2 计算方法

    comm中的task_online对其comm_analysis进行抽象。作为所有计算任务的基类

    每个计算任务类，只需要实现调用其相应算法的

    def Onlinecal(self):

    1.2 所有计算任务数据仿真基类

    comm_Simulation

    comm中的task_online对其comm_Simulation进行抽象。提供所有计算任务数据的简单随机仿真

    2 每个计算任务的实现
